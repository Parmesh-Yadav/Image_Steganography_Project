About Us

Steganography

Steganography is the technique of hiding secret data within an ordinary, non-secret, file or message in order to avoid detection; the secret data is then extracted at its destination.

Purpose

The purpose os this site is 1. To hide data in images.
2. Provide image ownership verification.
3. Provide image authenticity verification.
4. Generate invisible watermark.

Encryption

Data stored in the image is encrypted using AES-256 algorithm. Text is also compressed before encryption using huffman algorithm.

Decryption

Decrypted data is extracted from the image and will be mailed to the user. We do not store the sensitive data you encrypt.

Safety

To insure safety of data steps like encryption and compression are taken. Furthermore we do not store the sensitive data you encrypt.
